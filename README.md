# Parses Dominion (dominion.games) log to get decks


## Usage
Save your log into a text file and, run  
    `python dominion.py <log_path>`

To see what results from a sample game look like, just run  
    `python dominion.py`
    
For more help,  
    `python dominion.py --help`

if you find any.
